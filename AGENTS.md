# Agent Instructions

This repository is a legacy/reference working tree.

## Warning

- do not use this repository as the default target for current upstream-facing
  work
- for active server/core work, use `phorge`
- for active Arcanist and former libphutil-layer work, use `phorge-arcanist`
- only work directly in this repository if the user explicitly asks for it

## Purpose

- archaeology
- comparison against older behavior
- migration history and local reference

## Commit Guidance

- if exploratory fixes are made here for investigation, treat them as local
  notes unless the user explicitly wants them preserved
- before curating any patch for upstream-oriented work, verify whether the real
  fix location is `phorge-arcanist` instead
